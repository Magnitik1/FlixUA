# FlixUA — Full-Stack Streaming Platform

Netflix-inspired streaming platform built with React.js and Node.js.

## Features
- User authentication with server-side database validation
- REST API for content management
- Dynamic movie database integrated as backend data source
- Responsive UI with React Hooks and state management
- Custom video player (deployed separately)

## Tech Stack
React.js · Node.js · REST API · JavaScript · CSS

# FlixUA — Full-Stack Streaming Platform

A Netflix-inspired full-stack streaming platform built with React.js and Node.js.

## Features

- User authentication with server-side database validation
- REST API for content and user management
- Custom movie database designed and managed as a backend data source
- Dynamic content rendering via frontend–backend API integration
- Responsive UI built with React Hooks and state management
- Custom video player — deployed and accessible online

## Tech Stack

| Layer | Technologies |
|---|---|
| Frontend | React.js, React Hooks, Redux, CSS |
| Backend | Node.js, REST API |
| Database | Custom DB (movie storage + auth) |
| Tools | Git |

## Getting Started

```bash
git clone https://github.com/Magnitik1/NetflixClone.git
cd NetflixClone
npm install
npm start
```

App runs at `http://localhost:3000`
